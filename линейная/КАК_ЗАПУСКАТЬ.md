# Как запускать проект ft_linear_regression

Все команды выполняй в терминале из папки проекта (где лежат `train.py`, `predict.py`, `data.csv`).

---

## 1. Установка зависимостей (только для бонуса)

Для графиков нужен matplotlib. Если `pip` не находится, используй `python3 -m pip`:

```bash
python3 -m pip install matplotlib
```

Без matplotlib обязательная часть (train + predict) работает, просто не будет графика.

---

## 2. Обязательная часть

### Шаг 1: Обучить модель

```bash
python3 train.py
```

С флагом `--verbose` или `-v` выведутся промежуточные расчёты и порядок действий:

```bash
python3 train.py --verbose
```

Что произойдёт:
- Читается `data.csv`
- Считаются θ₀ и θ₁ градиентным спуском
- Результат сохраняется в `theta.csv`
- В консоль выводятся `theta0`, `theta1`

**Сначала всегда запускай обучение**, иначе в `theta.csv` не будет параметров (predict будет использовать 0 и 0).

### Шаг 2: Предсказать цену по пробегу

```bash
python3 predict.py
```

Что произойдёт:
- Программа спросит: `Enter car mileage (km):`
- Вводишь пробег в км (например, `100000`)
- Выводится строка вида: `Estimated price: 5610.00`

Можно запускать `predict.py` много раз с разными пробегами — модель уже обучена и лежит в `theta.csv`.

---

## 3. Бонусы

### График (данные + линия регрессии)

Запускается **автоматически** в конце `train.py`:

```bash
python3 train.py
```

В конце:
- Откроется окно с графиком (точки — данные, красная линия — регрессия)
- В папке проекта появится файл `plot.png` с тем же графиком

Если matplotlib не установлен, в консоль выведется сообщение, график не построится, но обучение и сохранение `theta.csv` пройдут нормально.

### Точность модели (R²)

**Вариант 1.** При обучении точность уже выводится в консоль:

```bash
python3 train.py
```

Ищи строку вида: `Precision (R²): 85.23%`

**Вариант 2.** Отдельная программа — посчитать точность по уже обученной модели (без повторного обучения):

```bash
python3 precision.py
```

Выведется одна строка, например: `Precision (R²): 85.23%`

Нужно, чтобы перед этим уже был запущен `train.py` и существовал файл `theta.csv`.

---

## Краткая шпаргалка

| Действие              | Команда           |
|-----------------------|-------------------|
| Установить matplotlib | `python3 -m pip install matplotlib` |
| Обучить модель        | `python3 train.py` |
| Обучить с подробным выводом | `python3 train.py --verbose` |
| Предсказать цену      | `python3 predict.py` |
| Посчитать точность    | `python3 precision.py` |

**Порядок при первом запуске:**  
`train.py` → потом `predict.py` или `precision.py` когда нужно.
